Navegadores web

Un navegador web (en ingles, web browser)9 es un software, aplicación o programa que permite el acceso a la web, interpretando la información de distintos tipos de archivos y sitios web para que estos puedan ser vistos

La funcionalidad básica de un navegador web es permitir la visualización de documentos de texto, posiblemente con recursos multimedia incrustados

Ejemplos de navegadores web

Safari: es un navegador web de código cerrado desarrollado por Apple Inc. Está disponible para macOS, iPadOS e IOS

Mozilla Firefox es un navegador web libre y de código abierto desarrollado para Linux, Andorid, IOS, macOS y Microsoft windows coordinado por la Corporación Umbrella y la Fundación Mozilla

Google: Google Chrome es un navegador web de código cerrado desarrollado por Google, aunque derivado de proyectos de código abierto
